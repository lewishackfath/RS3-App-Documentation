#Create an Event
The event scheduler allows to to create clan event listings which are displayed on the website, in a weekly summary discord post and in the daily event posts.

##Event Name
Enter the name for your clan event eg. "Amascut Hour".

###Event Date
Enter the date for your clan event, this field uses the local timezone configured for your clan.

###Event Time
Enter the time for your clan event, this field uses the local timezone configured for your clan.

###Start Time in UTC
You can use this field to pick the start date and time of your event in "Game Time" (UTC).

###Duration
Enter the event duration in minutes (Defaults to 60 minutes).

###Discords Channel Override
You can use this to override the default channel that will be used for the "Daily Event Post". Leave blank to use the default channel.

###Event Host
Select the discord user of the player who will be hosting the event (can be left blank if not required).

###Weekly Recurring Event
Use this checkbox if you want the event to occur weekly. If selected you will be prompted to select a date that the recurrence ends.

###Location
Enter the location of the event Eg. "World 12 - Grand Exchange".

###Create Voice Channel for Event
Enabling this will create a new voice channel for the event. The voice channel will only allow people who are subscribed to the event + the event host to speak in the channel by default. This is to ensure that the topics of discussion in the channel remain on the event. Anyone with Server mute permissions can allow others to speak if needed.

###Custom Image URL
Paste the link to an image you would like to use as the event icon. This is shown on the web page + in the discord daily event post.

###Notes/Description
Enter some notes or information about the event.

###Preferred Roles
You can use this to create Roles and insert emoji's to use for reactions. Eg. Amascut roles could be Base/Tank, Green 1, Green 2, East In, East Out.... etc.
